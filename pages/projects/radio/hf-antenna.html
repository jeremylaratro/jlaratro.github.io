STEALTH HF ANTENNA

// Multi-Band End-Fed Wire Antenna

OVERVIEW

Designed and built a stealth end-fed wire antenna for multi-band HF operation. The antenna is configured for optimal multi-band performance and hidden from view in an HOA-restricted area.

DESIGN CHALLENGE

Many amateur radio operators face restrictions from Homeowners Associations (HOAs) that prohibit visible antennas. This project addresses that challenge by creating an effective HF antenna that remains virtually invisible when installed.

End-fed wire antennas are ideal for stealth installations because they can be run along fence lines, through trees, or along roof edges where they blend into the environment.

TECHNICAL DETAILS

Type End-Fed Half-Wave
Bands Multi-Band HF
Matching 49:1 Transformer
Installation Stealth/Hidden

PERFORMANCE

Despite the compromised installation, the antenna performs remarkably well on multiple HF bands. The end-fed design with a 49:1 matching transformer allows for efficient power transfer and low SWR across the target frequencies.
