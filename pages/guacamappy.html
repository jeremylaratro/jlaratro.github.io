GuacaMappy is a cross-platform mobile client for Apache Guacamole,
enabling remote desktop access from iOS and Android devices.
Full support for RDP, VNC, SSH, and Telnet protocols with
enterprise-grade security features.

FEATURES

Multi-Protocol Support

Connect to remote systems via RDP, VNC, SSH, and Telnet - all through a single unified interface.

MFA/TOTP Authentication

Enterprise-grade security with multi-factor authentication and TOTP support for secure access.

WebSocket Tunneling

Secure WebSocket connections for reliable remote access through firewalls and NAT.

Touch-to-Mouse Translation

Intuitive touch controls mapped to mouse input for seamless desktop interaction on mobile devices.

Cross-Platform

Single codebase for iOS and Android using Flutter, ensuring consistent experience across devices.

SSH Terminal

Full-featured SSH terminal with keyboard support for command-line access to remote servers.

TECH STACK

Frontend

  • Flutter
  • Dart
  • Material Design

Protocols

  • RDP (Remote Desktop)
  • VNC
  • SSH / Telnet

Backend

  • Apache Guacamole
  • WebSocket API
  • guacd Protocol
