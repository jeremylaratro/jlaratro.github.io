SSTV DECODER

// Web-based Slow Scan TV Decoder

OVERVIEW

A web-based Slow-Scan Television (SSTV) decoder that processes audio input to decode amateur radio SSTV transmissions directly in the browser. No software installation required.

ABOUT SSTV

Slow Scan Television (SSTV) is a method of transmitting still images over radio frequencies. Unlike broadcast television, SSTV transmits images slowly, taking several seconds to several minutes per image. It's popular among amateur radio operators for exchanging QSL cards and other images.

SSTV encodes image data as audio tones, with different frequencies representing different brightness levels and colors. The audio can be transmitted over standard voice radio channels.

FEATURES

  • Browser-based operation - no installation needed
  • Real-time audio decoding
  • Support for common SSTV modes
  • Works with microphone input or audio files
  • Useful for decoding ISS transmissions

COMMON USE CASES

The International Space Station (ISS) occasionally transmits SSTV images that can be received with simple equipment. This decoder makes it easy to capture and decode these transmissions without installing specialized software.
