GrepEx is an iOS application designed for security researchers and
developers to generate regex patterns and Google search dorks.
Built with React Native and TypeScript for optimal performance.

FEATURES

80+ Pattern Templates

Pre-built regex patterns for common security research scenarios including IPs, emails, URLs, credentials, and more.

21 Language Outputs

Generate regex patterns formatted for Python, JavaScript, Go, Rust, Java, and 16 other programming languages.

GHDB Integration

Access to 3600+ Google Hacking Database dorks for vulnerability discovery and reconnaissance.

ReDoS Protection

Built-in detection and warnings for potentially vulnerable regular expressions that could cause denial of service.

Live Testing

Real-time regex testing against sample text with match highlighting and group extraction.

Gamification

Achievement system to track progress and encourage exploration of regex and dork capabilities.
