ThreatMobile is a mobile threat intelligence application providing
security analysts with real-time IOC lookups, threat feed aggregation,
and vulnerability tracking on iOS and Android devices.

FEATURES

IOC Lookups

Quick lookup of IP addresses, domains, file hashes, and URLs against multiple threat intelligence sources.

Threat Feed Aggregation

Real-time aggregation from VirusTotal, AlienVault OTX, ThreatFox, AbuseIPDB, and more.

CVE Tracking

Monitor CVE database for new vulnerabilities with severity scoring and exploit availability alerts.

Real-Time Alerts

Push notifications for new threats matching your watchlists and monitored indicators.

Watchlists

Create custom watchlists for domains, IPs, and hashes to monitor for new threat intel.

Cross-Platform

Single Flutter codebase for iOS and Android with native performance and offline support.

INTEL SOURCES

VirusTotal

File and URL analysis with 70+ AV engines.

AlienVault OTX

Community threat intelligence platform.

Shodan

Internet-connected device intelligence.

GreyNoise

Internet background noise analysis.

TECH STACK

Frontend

  • Flutter
  • Dart
  • Material Design 3

APIs

  • Threat Intel APIs
  • CVE Database
  • MITRE ATT&CK

Features

  • Push Notifications
  • Offline Cache
  • Biometric Auth
