AugFly is an augmented reality flight tracking application for iOS.
Point your camera at the sky to see real-time flight information
overlaid on aircraft using ARKit and live ADS-B data feeds.

FEATURES

ARKit Integration

Advanced augmented reality using Apple ARKit for precise aircraft overlay and tracking.

Real-Time ADS-B Data

Live aircraft tracking using ADS-B transponder data for accurate flight information.

GPS Location

Accurate positioning using device GPS for calculating aircraft positions relative to your location.

Flight Details

View airline, flight number, aircraft type, altitude, speed, origin, and destination in AR view.

Native iOS

Built with Swift and SwiftUI for optimal performance and native iOS experience.

Offline Caching

Aircraft and airline database cached locally for faster lookups and offline identification.

TECH STACK

Frontend

  • Swift
  • SwiftUI
  • ARKit

Data Sources

  • ADS-B Exchange
  • OpenSky Network
  • FlightAware API

Features

  • Core Location
  • SceneKit
  • Combine
